Qualia

A Name for himself. A Name to identify the kind of people he wants to fornicate with. A list of Names for the objects he touches daily. A list of Names of the parts of his body he touches the objects with.

A Name for every moment of the day. A Name for the activity to be done at a given moment. A Name for every color of the rainbow and every possible state of every fundamental particle.

He not only has a Name for the color I call Red, but also a Name for the Red-ness of the color I see when I see it, and all the name-able things that constitute Redness to me or to anyone. Of course he knows of, in order to name them, all these name-able things that constitute Redness. Pointers to pointers to pointers to pointers. He has all the syntax there is in the universe.

He has a name for all name-able things, name-able sub-things of things and name-able processes through which all these things interact and nothing except a name. He has a name for all things that can be named which must be all things, must it not? If he finds a new thing, he will name it and name all the processes and interactions through which the thing participates with the world he has already catalogued.
